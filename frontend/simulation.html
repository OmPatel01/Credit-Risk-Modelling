📚 What is Monte Carlo Simulation?

Monte Carlo Simulation:
Generates thousands of random scenarios where borrowers either default or repay. For each scenario, calculates total portfolio losses. Returns a distribution of possible losses.
Expected Loss (EL):
Average loss across all simulations. This is the mean of the loss distribution.
Unexpected Loss (UL):
Variability in losses (standard deviation). Higher UL means more unpredictability.
Value at Risk (VaR):
Maximum loss at 95% confidence level. Example: "95% VaR = $100K means worst case loss in 95% of scenarios is $100K."
Conditional VaR (CVaR):
Average loss beyond VaR. Captures tail risk — what happens in worst 5% of scenarios.

📌 Model Assumptions
  • Probability of Default (PD) is derived from risk level slider
  • Loss Given Default (LGD) is assumed constant at 50%
  • Exposure at Default (EAD) is based on portfolio loan amounts
  • Defaults are simulated assuming borrower independence
  • Monte Carlo simulation generates random default scenarios
