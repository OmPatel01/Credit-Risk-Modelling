Sensitivity Analysis

Identify key risk drivers and measure impact on portfolio losses

📚 What is Sensitivity Analysis?

Purpose:
Measures how changes in key risk drivers (PD, LGD) impact total portfolio losses. Identifies which factors have the greatest influence on risk.
PD Sensitivity
Measures how portfolio losses change when default probability shifts by ±10%, ±20%.

Formula: Δ ECL = Δ PD × LGD × EAD

LGD Sensitivity
Measures how portfolio losses change when loss-given-default shifts by ±5%, ±10%.

Formula: Δ ECL = PD × Δ LGD × EAD

Key Insights
Identify which drivers have the greatest impact:
  • Elasticity: % change in loss per 1% change in driver
  • Allows for targeted risk mitigation

🔍 Run Sensitivity Analysis

📌 Sensitivity Assumptions
  • Baseline PD is assumed constant across portfolio (default: 10%)
  • Baseline LGD is fixed at 50%
  • Exposure at Default (EAD) is derived from loan amounts
  • PD is varied using predefined shifts (e.g., ±10%, ±20%)
  • LGD is varied independently using predefined shifts (e.g., ±5%, ±10%)
  • Other variables are held constant (ceteris paribus)
