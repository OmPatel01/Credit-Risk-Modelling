Evaluate portfolio resilience under adverse economic scenarios

📚 Stress Scenarios

🟢 Base Case

Scenario: Normal economic conditions

• Interest rates: stable
• Employment: stable
• PD multiplier: 1.0x

🟡 Mild Stress

Scenario: Economic slowdown

• Interest rates: rising
• Employment: slight decline
• PD multiplier: 1.5x

🔴 Severe Stress

Scenario: Economic recession

• Interest rates: high
• Employment: significant decline
• PD multiplier: 3.0x

⚠️ Run Stress Test

📌 Stress Test Assumptions
  • Base PD is derived from portfolio (default: 10%)
  • Stress scenarios apply PD multipliers (e.g., 1.5x, 3.0x)
  • LGD is adjusted per scenario or assumed constant at 50%
  • EAD is based on loan amounts in the portfolio
  • Macroeconomic stress is simulated via scaling factors (PD/LGD)
  • Optional Monte Carlo simulation assumes independent defaults
